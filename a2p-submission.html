Prerequisites (10DLC - EIN Clients)

Before starting 10DLC registration:

  • LOCAL number purchased in sub-account (area code matching client location)
  • Website homepage is LIVE (verification requires live URL)
  • Client onboarding form completed (need business details)
  • EIN letter on file (for exact legal name/address match)
No EIN? Skip to Toll-Free Verification instead.

Step 1: Access 10DLC Registration

  1. Go to client sub-account
  2. Settings → Phone Numbers
  3. Find the local number
  4. Click "Register for A2P 10DLC"

Step 4: Submit and Track

  2. Submit campaign
  3. Note submission date in client record
  4. Add tag a2p-pending to contact in agency account

Expected timeline: 2-4 business days for brand, 1-2 days for campaign

Toll-Free Verification (No-EIN Clients)

For sole proprietors without an EIN, use toll-free verification instead of 10DLC:

Prerequisites

  • TOLL-FREE number purchased (888, 877, 866, etc.)
  • Website homepage is LIVE
  • Client onboarding form completed

Process

  1. Go to client sub-account → Settings → Phone Numbers
  2. Find the toll-free number
  3. Click "Verify this Toll Free Number"
  4. Enter business information (legal name can match what client provided - no EIN verification)
  5. Enter website URL
  6. Describe use case: "Customer care and appointment reminders for [business type] services"
  7. Submit

Expected timeline: 1-3 business days (faster than 10DLC)

If Rejected

Common rejection reasons:

  1. Legal name mismatch - Must match EIN exactly
  2. Address mismatch - Must match EIN exactly
  3. Missing consent checkbox - Website form needs visible checkbox
  4. Privacy policy issues - Must be live and accessible

To fix:

  1. Open GHL support ticket via Zoom
  2. Get ticket number
  3. Book with Gabby: https://speakwith.us/gabbyc

Update tags:

  • Add a2p-rejected
  • Note rejection reason

Escalate to Dan if:

  • Client threatening to cancel over A2P delays

After Approval

  1. Remove a2p-pending tag
  2. Add a2p-approved tag
  3. Test SMS sending from sub-account
  4. Move client to next stage if blocked on A2P
