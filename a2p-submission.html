A2P 10DLC Submission SOP

Role: Fulfillment Specialist
When: After toll-free number purchased, website homepage live
Time: 10-15 minutes per submission

Prerequisites

Before starting:

  • Toll-free number purchased in sub-account

  • Website homepage is LIVE (verification requires live URL)

  • Client onboarding form completed (need business details)

  • EIN letter on file (for exact legal name/address match)

Step 1: Access A2P Registration

  1. Go to client sub-account
  2. Settings → Phone Numbers
  3. Find the toll-free number
  4. Click "Verify this Toll Free Number"

Step 2: Brand Registration

Enter information exactly as shown:

Submit brand registration. Wait for approval (usually 1-2 days).

Step 4: Submit and Track

  1. Review all information
  2. Submit campaign
  3. Note submission date in client record
  4. Add tag a2p-pending to contact in agency account
Expected timeline: 2-4 business days

If Rejected

Common rejection reasons:

  1. Legal name mismatch - Must match EIN exactly

  2. Address mismatch - Must match EIN exactly

  3. Missing consent checkbox - Website form needs visible checkbox

  4. Privacy policy issues - Must be live and accessible


To fix:
  1. Open GHL support ticket via Zoom

  2. Get ticket number

  3. Book with Gabby: https://speakwith.us/gabbyc

  4. She will NOT accept without ticket number


Update tags:
  • Remove a2p-pending

  • Add a2p-rejected

  • Note rejection reason


After Approval

  1. Remove a2p-pending tag
  2. Add a2p-approved tag
  3. Test SMS sending from sub-account
  4. Move client to next stage if blocked on A2P

Checklist

  • Prerequisites verified
  • Legal name matches EIN exactly
  • Address matches EIN exactly
  • Brand registration submitted
  • Brand approved
  • Campaign registration submitted
  • Tag updated (a2p-pending)
  • Approval received / Rejection handled
  • Tag updated (a2p-approved or a2p-rejected)
  • SMS tested
