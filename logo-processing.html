Scenario 1: Client Uploaded Logo

Step 1: Download Logo

  1. Go to client's onboarding form response
  2. Find logo upload field
  3. Download the file
  4. Check file format: PNG, JPG, SVG, or PDF acceptable

Step 2: Verify Quality

Check:

  • Resolution: Minimum 400x400 pixels
  • Background: Transparent preferred (PNG/SVG)
  • Not blurry or pixelated
  • Colors look correct

If low quality:

  • Request higher resolution from client
  • Or recreate in Canva (see Scenario 2)

Step 3: Resize for Website

Website needs logo at 400x400 pixels or similar square ratio.

Using Canva:

  1. Create new design: Custom size 400x400
  2. Upload client logo
  3. Center and scale to fit
  4. Download as PNG (transparent background)

Using Preview (Mac):

  1. Open logo file
  2. Tools - Adjust Size
  3. Set to 400x400 (uncheck "Scale proportionally" if needed)
  4. Save

Step 4: Upload to GHL Media Library

  1. Go to client sub-account
  2. Sites - Media Library
  3. Upload resized logo
  4. Copy the URL
  5. Paste into Custom Values - logo_url

Scenario 2: Create Logo in Canva

When client checked "needs logo" or uploaded something unusable.

Step 1: Gather Info

From onboarding form:

  • Business name
  • Industry/niche
  • Any color preferences
  • Any style preferences

Step 2: Open Canva

  1. Go to canva.com
  2. Create new design - Logo (500x500 recommended)

Step 3: Find Template

  1. Search templates by industry (e.g., "plumbing logo", "landscaping logo")
  2. Pick 2-3 that match the vibe
  3. Choose one that's clean and professional

Step 4: Customize

Text:

  • Replace with business name
  • Keep it readable
  • Don't use more than 2 fonts

Colors:

  • Use color_primary from their custom values
  • Or use industry-appropriate colors
  • Keep it simple (2-3 colors max)

Icon:

  • Choose relevant icon for their trade
  • Don't overcomplicate

Step 5: Export

  1. Click Share - Download
  2. File type: PNG
  3. Enable "Transparent background"
  4. Download

Step 6: Upload to GHL

Same as Scenario 1, Step 4.

Logo Style Guidelines

DO:

  • Keep it simple and clean
  • Make text readable at small sizes
  • Use professional colors
  • Include an icon related to their trade
  • Ensure it works on white AND dark backgrounds

DON'T:

  • Use more than 2 fonts
  • Make it overly complex
  • Use clip art that looks cheap
  • Include taglines (too small to read)
  • Use gradients that don't scale well

Quality Checklist

Before uploading final logo:

  • Resolution: 400x400 minimum
  • Transparent background (PNG)
  • Readable at small sizes (favicon test)
  • Looks good on white background
  • Looks good on dark background (if used in header)
  • Business name spelled correctly
  • Not pixelated or blurry

Storage

Save original + processed logos to:

  • Client's folder in Google Drive (if using)
  • Or GHL Media Library only

Troubleshooting

Logo looks bad on website

  1. Check if header background is dark vs light
  2. May need two versions (light and dark)
  3. Adjust logo_url to appropriate version

Client doesn't like logo we created

  1. Ask what specifically they don't like
  2. Make 2-3 variations
  3. Let them choose
  4. If still unhappy: Ask them to provide their own

Logo file is PDF

  1. Open PDF in Preview (Mac) or online converter
  2. Export as PNG
  3. Process as normal
