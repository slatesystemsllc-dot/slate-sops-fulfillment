Overview

This SOP covers connecting and configuring a client's Google Business Profile (GMB) to their Slate Systems sub-account, plus all reputation management setup.

Prerequisites:

  • Client has granted manager access to GMB profile (via guides.slatesystems.io instructions)
  • Phone number purchased and configured (local or toll-free)
  • Website live on custom domain

Step 1: Connect GMB to GHL (5 min)

Get Client Access

  1. SMS sent automatically: WF-OB3 sends GMB access request with guides.slatesystems.io link
  2. Client action: They follow guide to grant manager access to hello@slatesystems.io
  3. Verify: Check agency Gmail for invitation, accept it

Connect in GHL

  1. Switch to client sub-account
  2. Go to Settings - Integrations
  3. Click Sign in with Google
  4. Use hello@slatesystems.io (agency account)
  5. Select client's business from the list
  6. Click Connect

Troubleshooting:

  • If business not visible: Client didn't complete access grant - resend guides link
  • Multiple locations: Select the correct one (verify address)
  • Connection fails: Try disconnecting agency Google, reconnect

Step 2: Update GMB Profile (5 min)

Open Google Business Profile Manager

  1. Go to business.google.com
  2. Find client's business
  3. Click Edit Profile

Update Phone Number

  1. Find Phone number section
  2. Replace with purchased phone number: (XXX) XXX-XXXX
  3. Remove any secondary numbers

Update Website

  1. Find Website section
  2. Enter: https://www.[domain].com
  3. Always include www. prefix
  4. Remove old website if present

Verify Business Hours

  1. Check Business hours section
  2. Compare with client_hours from onboarding form
  3. Update if different

Save Changes

  1. Click Save
  2. Changes may take 24-48 hours to appear publicly

Step 3: Get GMB Values for Custom Values (5 min)

Company GMB Link

  1. In Business Profile Manager, view the business profile
  2. Copy the full URL from browser address bar
  3. Enter in sub-account: Settings - Custom Values - company_gmb_link

GMB Review Link

  1. In Business Profile Manager, click Ask for reviews
  2. Copy the short link provided (looks like https://g.page/r/[ID]/review)
  3. Enter in ALL THREE of these custom values:
  • company_google_review_link - used in emails/SMS
  • review_request_link - used in review request workflows
  • review_google_url - CRITICAL: used by Review Survey to redirect 4-5 star ratings to Google
WARNING: Must use the g.page/r/.../review format from "Ask for reviews". Do NOT use regular Google Maps links (maps.app.goo.gl/... or google.com/maps/place/...) - those just show the business page, not the review form. The /review suffix opens the review dialog directly.

Google Map Embed

  1. Go to Google Maps
  2. Search for client's business name
  3. Click Share - Embed a map
  4. Copy the <iframe> code
  5. Modify dimensions: width="450" height="400"
  6. Enter in: Custom Values - google_map_embed

Step 4: Verify Reviews AI (3 min)

Reference: See slate-review-ai-agent.md for full agent configuration

The Reviews AI agent should already exist from the Carlsbad snapshot. Verify it's configured:

  1. Go to Reputation - Settings - Reviews AI
  2. Verify agent "Positive Review Responder" exists
  3. Verify settings:
  • Review Type: 4-5 stars only
  • Tone: Professional + Grateful
  • Auto-response: Enabled
  1. Verify GMB is connected as a source

If missing: Create per slate-review-ai-agent.md

Step 5: Configure Social Planner (3 min) - REQUIRED

What this does: Automatically posts Google reviews AS Google Posts on the client's GMB profile. This helps with SEO and showcases best reviews directly on their Google listing.
Why this matters: Every two reviews they get weekly will automatically get published to their Google page, boosting SEO and visibility.

Do this for every client with existing Google reviews.

Navigation

  1. Go to Marketing - Social Planner
  2. Click Content tab (NOT "Planner" tab)
  3. Click Review button
  4. Click New Post

Configuration

  1. Select Review Source:
  • Source: Google (check)
  • Facebook: optional if client has FB page
  1. Star Filter:
  • Select: 5 stars only (we only post best reviews)
  1. Select Destination:
  • Check: Google My Business (client's GMB profile)
  • Can also add Facebook/Instagram if client has them
  1. Frequency:
  • Select: Weekly (posts ~2x per week)
  • Do NOT use daily - too spammy, diminishing returns
  • Best practice: Post reviews about two times per week for optimal engagement
  1. Design:
  • Choose background images (can select multiple - system rotates)
  • Leave caption blank or customize
  • Display author name: Yes
  • Author display: First name or initials
  1. Click Create Post

Verify Setup

  • Go to Social Planner - Scheduled
  • Confirm review posts are queued
  • First posts will appear within 24-48 hours

Skip only if: Client has zero existing Google reviews (configure later when they get their first reviews)

Step 6: Create Review Widget (3 min) - REQUIRED

  1. Go to Reputation - Widgets - Create New
  2. Configure:
  • Name: "Homepage Reviews"
  • Layout: Carousel
  • Review Filter: 5 Star Only
  1. Appearance:
  • Primary Color: #ffffff (white)
  • Secondary Color: Transparent
  • Background Color: Transparent
  • Border Color: Transparent
  1. Settings:
  • Check "Exclude reviews with no description"
  1. Click Save - Make it LIVE
  2. Copy the embed code
  3. Go to Sites - [Website] - Edit Homepage
  4. Find Reviews section - paste widget embed code

Step 7: Add Schema Markup (2 min) - REQUIRED

  1. Go to Sites → [Website] → Settings
  2. Find Head Tracking Code section
  3. Choose @type based on niche:
    • Painters, Roofers, General Contractors: HomeAndConstructionBusiness
    • Plumbers: Plumber
    • HVAC: HVACBusiness
    • Electricians: Electrician
    • Landscapers: LandscapingBusiness
    • Other: LocalBusiness
  4. Add this code (replace [TYPE] with chosen type):
<script type="application/ld+json">{
  "@context": "https://schema.org",
  "@type": "[TYPE]",
  "name": "{{custom_values.company_name}}",
  "url": "{{custom_values.company_website_link}}",
  "telephone": "{{custom_values.company_twilio_phone}}",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "{{location.address}}",
    "addressLocality": "{{location.city}}",
    "postalCode": "{{location.postal_code}}",
    "addressCountry": "{{location.country}}"
  }
}</script>
  1. Click Save

Why: Helps Google understand the business for local SEO

  • google_map_embed filled (450x400 dimensions)

Reputation Features

  • Map embed displays correctly on website

Troubleshooting

Client Doesn't Have GMB

  1. Send them to guides.slatesystems.io - "Create Your Profile" section
  2. Guide them through verification (video preferred for speed)
  3. Verification typically takes 1-14 days
  4. Note in pipeline: "Pending GMB Verification"
  5. Continue build - GMB values can be added later

GMB Claimed by Someone Else

  1. Ask client if they know who owns it
  2. If not: Guide through Google's recovery process
  3. Send them to guides.slatesystems.io - "Troubleshooting" section
  4. Recovery takes 7+ days
  5. Continue build without GMB

Map Embed Not Showing

  1. Verify iframe code copied completely
  2. Check dimensions are 450x400
  3. Preview website to verify display
  4. Try regenerating embed code from Google Maps

Social Planner Not Posting

  1. Verify GMB account is connected
  2. Check that client has 5-star reviews
  3. Verify frequency is set correctly
  4. Check Social Planner - Scheduled for upcoming posts

Escalation

Escalate to Dan if:

  • GMB recovery stuck for 10+ days
  • Client frustrated about GMB delays - before they threaten cancellation
  • Google suspends or removes client's GMB profile
  • Reviews AI not responding to reviews after 48 hours

Appendix: Five-Star Review Funnel System

Purpose: This "review protection" system filters feedback to maximize positive public reviews.

Overview

The review funnel ensures clients ONLY get public Google reviews from happy customers. Unhappy customers vent privately.

Components

ComponentLocationPurpose
Review Funnel PageSites - Website - /review-usStar selector (1-5)
SurveySites - SurveysPrivate feedback form (1-3 stars)
WorkflowAutomations - "negative feedback received"Notifies client of bad feedback

 Feedback Form]         Google Reviews]
         |                   |

 feedback to client     posted on GMB]
 via SMS]

Setup Checklist

Review Funnel Page:

  • Page exists at /review-us or /leave-review
  • 5 star buttons visible
  • 4-5 star buttons link to {{custom_values.company_google_review_link}}
  • 1-3 star buttons redirect to Survey

Survey (Private Feedback):

  • Survey exists in Sites - Surveys
  • Has text field for feedback
  • Styled to match website
  • Thank you message displayed after submit

Workflow:

  • "negative feedback received" workflow exists
  • Triggers on Survey submission
  • Sends SMS to client with feedback content
  • Does NOT post to Google

Verify in Carlsbad

Before client launch, confirm:

  1. /review-us page exists and star buttons work
  2. Survey form captures feedback
  3. Test 1-3 star flow - confirm client gets SMS
  4. Test 4-5 star flow - confirm redirect to Google
